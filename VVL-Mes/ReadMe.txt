
->> Задание:
    Написать мессенжер (не чат, а именно мессенжер) с графическим интерфейсом пользователя.

    Приложение поддерживать следующие действия:
    1. Отображать список контактов пользователя (или список доступных в сети пользователей).
    2. Отображать поле ввода для отправки сообщения выбранному пользователю (с самой отправкой, разумеется).
    3. Переписка двух пользователей не должна быть доступна кому-то третьему.


->> Описание мессенджера:
    Название мессенджера - VVL-Messenger
    Работа с мессенджером:
        1. Для входа в систему нужно ввести имя пользователя в поле Your name и нажать кнопку ) и Имя получателя (Receiver).
        2. Сообщение ввести в нижнем блоке (Введите текст...) и отправить с помощью кнопки Send.
        3. После отправки сообщение будет выведено в блоке слева (самый большой блок окна интерфейса).
        В блок вывода добавляются только те сообщения, которые отправляли пользователи, указанные в блоках:
        Имя отправителя (Your name) и Имя получателя (Receiver)
        4. В блоке слева (Recipient list) выводится Список контактов для Оправилеля, указанного в блоке (Your name).


->> Запуск мессенджера:
    1. Из этой папки все файлы перенести в виртуальное окружение.
    2. Установить в виртуальном окружении библиотеки flask, requests, PyQt5.
        pip install flask
        pip install requests
        pip install PyQt5
    3. Запустить файл server.py для создания локального сервера.
    4. Запустить файл messenger.py для открытия интерфейса.
