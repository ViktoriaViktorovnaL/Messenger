
->> Задание:
    Написать мессенжер (не чат, а именно мессенжер) с графическим интерфейсом пользователя.

    Приложение поддерживать следующие действия:
    1. Отображать список контактов пользователя (или список доступных в сети пользователей).
    2. Отображать поле ввода для отправки сообщения выбранному пользователю (с самой отправкой, разумеется).
    3. Переписка двух пользователей не должна быть доступна кому-то третьему.


->> Описание мессенджера:
    Название мессенджера - VVL-Messenger
    Работа с мессенджером:
        1. Для начала работы нужно ввести URL сервера в верхнем поле и нажать кнопку Ok.
            Если при взаимодействии с публичным сервером возникнит ошибка, приложение запустится на локальном сервере.
        2. Для входа в систему нужно ввести имя пользователя в поле Your name и нажать кнопку Sign in, для выхода нажать Sign out.
            После входа в систему в блоке слева (Recipient list) выводится Список контактов.
            Список контактов введенный мной отображается в этом поле в виде замещаемого текста.
            Имя вашего пользователя может быть новым, тогда Мессенджер добавит его в контакты.
        3. Для отправки сообщение:
            - заполнить поле Имя получателя (Receiver);
            - ввести текст сообщения в нижнем блоке (Введите текст...);
            - затем нажать кнопку Send.
        

->> Запуск мессенджера:
    1. Из этой папки все файлы перенести в виртуальное окружение.
    2. Установить в виртуальном окружении библиотеки flask, requests, PyQt5.
        pip install flask
        pip install requests
        pip install PyQt5
    3. Запустить файл server.py для создания локального сервера.
    4. Запустить файл messenger.py для открытия интерфейса.
